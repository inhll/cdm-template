Using The CDM D2L Template

I've created a template for CDM to help you get started building in D2L really quickly and without a lot of headache, however you are not required to use this template.

This template is constructed using Foundation 4, a responsive CSS/JS framework which makes it easy to create responsive course materials.

What's Included?

Instructor Only Files Cannot be seen by students.

These files (Fig. 1 in Red) contain information useful to you when building and running your course. There are four files here, the first of which, Using This Template, you are reading now. The included files are:

  • Using This Template explains what is included in this template.
  • A Quick D2L Overview describes the absolute basics of D2L's features.
  • Working with Foundation explains how to use Foundation, and provides example layouts.
  • Start of Term Checklist explains how to get your D2L course rolling.
  • Course Developer and Resources explains how to get in touch with me, what I can help with, and how to submit feature requests.

Sample Module is an example module where you can experiment and also includes notes describing the type of content intended for each section and it's pedagogical value.

Getting Started Module Visible to students.

This module (Fig. 1 in Green) includes information to help your students get started in the course. Providing a quick introduction and explaining what is expected can go a long way toward keeping student confusion (and therefore, email) to a minimum.

Modules 1-10 Visible to students.

There are 10 modules here (Fig. 1 in Blue) by default. Remove modules and sub-sections you don't want, add some that you do want, and make the modules your own. I'm just providing a scaffold for you to get started in D2L.
