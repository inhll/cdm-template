Module Title

Learning Objectives

Section Context

Learning objectives articulate the knowledge and skills you want students to acquire by the end of the module or course in measurable terms. The measurable terms should be related to the assessment strategy, for example:

  • being able to "identify the new HTML 5 tags" would be assessed by providing students a list and asking them to "identify" the correct items,
  • which is different from "describe the new HTML 5 tags" which would involve writing to "describe."

To learn about learning objectives, you may refer to the Teaching Commons. For assistance developing learning objectives, contact Ian Hall (ihall@depaul.edu).

Upon completing this module, you will be able to:

  • Identify key terms from [the materials covered].
  • Recall some stuff
  • Order some steps in a process
