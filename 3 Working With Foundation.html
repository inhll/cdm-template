Working with Foundation

Foundation Front-End Framework (CSS/JS)

This template uses Foundation, a responsive front-end framework, to make it easier to format content in D2L. For your convenience both the CSS and Javascript libraries have been included. JS has been omitted from the default page layout below but implementing it is simple, just refer to the implementation documentation. Foundation is built into every page; you can use the following body for your HTML page:

<!DOCTYPE html>
<head>
  <meta charset="utf-8" />
  <!-- Set the viewport width to device width for mobile -->
  <meta name="viewport" content="width=device-width" />
  <link rel="stylesheet" href="../_css/normalize.css">
  <link rel="stylesheet" href="../_css/foundation.css">
</head>
<body>
  <div class="row">
   <div class="large-12 columns">

     <h1> [Your Page Title Goes Here] </h1> <hr />

     <p> [Your Page Content Goes Here] </p>

   <hr /><hr />
   </div>
  </div>
</body>
</html>

For more information on working with Foundation, visit the Foundation Documentation.
